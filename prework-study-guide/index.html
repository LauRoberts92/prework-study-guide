HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • When working with HTML elements, the content is contained between opening and closing 'tags' which are often represented by angle brackets.
  • Closing tags begin with a forward slash.
  • The H1, H2, H3, H4, H6 and H6 elements represent the level of heading (or size) a given text block represents.
  • H1 is the largetest element and should only be used once on a page.
  • The 'p' element represents a paragraph or a block of text.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Do Not Repeat Yourself (DRY) Coding is a codin principle that encourages developers to re-use or share peices of code to reduce the number of lines of code that need to be written or maintained.
  • DRY can greatly reduce the overall size and complexity of a codebase.

Git

  • git status: checks what branch we are currently on.
  • git checkout -b branch-name: creates a new branch and switches to it.
  • git merge : merges the specified branch history with the current one.
  • git push [alias] [branch-name] : transmit local branch commits to the remote repository branch.
  • git pull: fetch and merge any commits from tracking remote branch.

JavaScript

  • Control flow is the order in which a computer executes code in a script.
  • A variable is a named container that allows us to store data in our code.
  • Comments are snippets of text that can be added along with code. The browserignores text marked as comments.
  • An Operator is a mathematical symbol that produces a result based on two values or variables.
  • Events are code structures that listen to activity in the browser and run code in respose. e.g a click event.
  • A Console refers to a test environment that developers use to test their code.
  • An Array is a single variable that is used to hold a group of data. The data is usually related in some way.
